## Overview

GOATMOUTH is a web-based prediction market built specifically for Jamaicans.  
Users make predictions on real-world events including local news, politics, sports, culture, and trends.

Markets resolve based on real outcomes.

---

## Technology

- HTML  
- CSS  
- JavaScript  
- Supabase  

No frameworks. No blockchain dependency.

---

## Payments

Planned payment integrations include:

- Lynk  
- WiPay  
- Cryptocurrency  

All payments are designed to support Jamaican users first.
